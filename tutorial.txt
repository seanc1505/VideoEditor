Simple work flow for this program
1. Import source video of .mp4 format (New source video button)
2. Press the New Subclip Button
        -This will open a new subclip window 
3. Here add the start/duration/end times of each clip
        -Only one time entry is required. 
        -Start and end time will be prioritised if all three times are entered
4. Press create subclip
        -This will display the start and end time and frames of the clip below
        -It will reset the times boxes
        -It will increment the clip number
5. Repeat steps 3/4 with as many subclips as desired
        -Replacing one subclip can be done by changing the subclip number in the box
6. Press close and all of these subclips will be stored
7. If clips from another source video are desired add a new source video (step1 onwards)
8. When all subclips have been set, enter the export video detail names and press set export name
9. Press export video and allow the program some time to run
